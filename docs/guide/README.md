# Guide
Owner: Backend Team
Last reviewed: 2026-02-04

## Цель
Собрать материалы для быстрого старта и ежедневной разработки.

## Контекст
Раздел ориентирован на разработчиков и онбординг. Здесь только практические шаги и правила работы.

## Шаги
1. Начните с быстрого старта.
2. При необходимости пройдите полный локальный сетап.
3. Используйте список скриптов и правила разработки как справочник.

## Содержание
- [Быстрый старт](quickstart.md) — минимальный запуск API.
- [Локальная настройка](local-setup.md) — полный процесс локального развёртывания.
- [Скрипты проекта](scripts.md) — команды npm.
- [Тестирование](testing.md) — как запускать unit и e2e тесты.
- [Правила разработки](development-rules.md) — правила качества и DoD.

## Ограничения
Раздел не описывает production-эксплуатацию и архитектурные решения. Для этого используйте Ops и ADR.
