# Деплой: Docker

Документ описывает доступные Docker-артефакты и сценарии запуска.

## docker-compose

В репозитории есть `docker-compose.yml` для запуска PostgreSQL локально. Он не включает контейнер приложения.

Пример запуска:

```bash
docker-compose up -d
```
