# Асинхронные задачи

Документ описывает фоновые задачи, очереди и cron-операции, если они есть в проекте.

## Текущее состояние

- В коде нет очередей Bull/BullMQ, cron-джоб или event emitter.
- Есть отдельный CLI процесс импорта адресов, который запускается вручную.

## CLI импорт адресов

Импорт запускается командой `npm run import:addresses:dev` или `npm run import:addresses` и не поднимает HTTP сервер.
