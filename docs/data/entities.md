# Данные: сущности

Документ описывает сущности TypeORM, используемые приложением.

## AddressEntity

Таблица: `addresses`.

**Назначение:** хранение нормализованных данных адресов и строк для поиска.

**Ключевые поля:**

- `id` — уникальный идентификатор адреса из источника.
- `x`, `y` — координаты (nullable).
- `displayKo`, `displayEn` — отображаемые строки.
- `searchKo`, `searchEn` — поисковые строки.
- Блоки `road*` и `parcel*` — детали дорожного и земельного адресов.

**Связи:** отсутствуют.
