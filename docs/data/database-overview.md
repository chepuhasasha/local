# Данные: обзор базы

Документ описывает используемую СУБД и высокоуровневую структуру таблиц, которую создаёт импорт адресов.

## Технология

- PostgreSQL используется как единственная БД.
- ORM: TypeORM.
- Подключение формируется на основе переменных окружения.

## Таблицы, создаваемые импортом

Импорт создаёт таблицы при первом запуске:

- `addresses` — текущий слепок адресов.
- `addresses_next` — теневой слепок для импорта.
- `addresses_prev` — предыдущий слепок.
- `address_import_state` — состояние импорта (месяц, статус, ожидания).

## Индексы

- Для поиска используются GIN индексы по `search_ko` и `search_en` с `pg_trgm`.
