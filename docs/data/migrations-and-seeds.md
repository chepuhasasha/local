# Данные: миграции и сиды

Документ описывает, как управляются миграции и наполнения базы.

## Текущее состояние

- В репозитории нет файлов миграций.
- Таблицы создаются на лету в CLI импорте адресов.
- `TypeORM` настроен на поиск миграций в `dist/infrastructure/database/migrations`, но каталог отсутствует.

## Что использовать вместо миграций

Для инициализации БД нужно запускать импорт адресов, который создаёт таблицы и индексы.
