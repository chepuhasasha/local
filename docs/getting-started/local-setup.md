# Локальная настройка

Документ описывает системные требования и подготовку окружения для локального запуска сервиса.

## Требования

- Node.js 18+.
- PostgreSQL 14+.

## Подготовка окружения

1) Скопируйте пример переменных окружения:

```bash
cp .env.development.example .env
```

2) Запустите PostgreSQL:

```bash
docker-compose up -d
```

3) Импортируйте адреса, чтобы создать таблицы и индексы:

```bash
npm run import:addresses:dev
```

4) Запустите API:

```bash
npm run start:dev
```
