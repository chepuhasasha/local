# Интеграции: внешние сервисы

Документ фиксирует внешние зависимости, с которыми взаимодействует сервис.

## Загрузка архивов адресов

CLI импорт загружает архив с данными адресов по HTTP(S):

- Базовый URL формируется в коде:
  - `https://business.juso.go.kr/api/jst/download` с query-параметрами.
- Параметры формируются на основе `ADDRESS_DATA_MONTH`.

Дополнительные настройки не предусмотрены (API ключи не используются в коде).
